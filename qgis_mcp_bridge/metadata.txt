about=Meridian starts a small HTTP server bound to 127.0.0.1 inside QGIS and executes PyQGIS on the Qt main thread. Paired with the companion MCP server (shipped in the source repository), it lets an AI assistant such as Claude programmatically control QGIS: execute PyQGIS code, manage layers, run Processing algorithms, and render the map canvas to an image. SECURITY: this plugin enables arbitrary code execution by design. It binds to localhost only, refuses cross-origin (browser) requests, and supports an optional shared-secret token (QGIS_MCP_TOKEN). Use only with trusted, local MCP clients. Distributed via GitHub (Install from ZIP), not the official plugin repository.

changelog=0.1.0
    - Initial release as "Meridian".
    - Localhost HTTP bridge that executes PyQGIS on the QGIS main thread.
    - Endpoints: /ping, /execute (arbitrary PyQGIS), /render (canvas PNG).
    - Hardening: cross-origin/Referer requests refused; optional QGIS_MCP_TOKEN shared secret.
    - Menu toggle to start/stop the bridge; autostarts on load.
