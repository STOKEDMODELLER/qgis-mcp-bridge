about=QGIS MCP Bridge starts a small HTTP server bound to 127.0.0.1 inside QGIS and executes PyQGIS on the Qt main thread. Paired with the companion MCP server (shipped in the source repository), it lets an AI assistant such as Claude programmatically control QGIS: execute PyQGIS code, manage layers, run Processing algorithms, and render the map canvas to an image. The server listens on localhost only. Note: this plugin enables arbitrary code execution by design and is intended for trusted, local automation.
