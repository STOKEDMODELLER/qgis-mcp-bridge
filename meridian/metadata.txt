about=Meridian runs a small HTTP server bound to 127.0.0.1 inside QGIS and dispatches a fixed, reviewed set of commands to PyQGIS on the Qt main thread. Paired with the companion MCP server (in the source repository), it lets an AI assistant such as Claude operate QGIS through vetted operations: inspect the project, add/remove layers, read features, run named Processing algorithms, style, zoom, and render the map. There is no eval/exec — only specific handlers. The server binds to localhost only, refuses cross-origin (browser) requests, and supports an optional shared-secret token (QGIS_MCP_TOKEN).
